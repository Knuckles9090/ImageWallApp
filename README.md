DEMO
---------------------------------
http://imagewallapp.herokuapp.com

Setup
---------------------------------
- 1. "run npm install"
- 2. "npm start"

Developing
----------
Run "nodemon" to have your files watched

Building distribution
---------------------------------
To build dist folder, use
"npm run build"

Running tests
---------------------------------
Run unit tests with
"npm run unit-tests"

Run e2e tests with
"npm run e2e-tests"
